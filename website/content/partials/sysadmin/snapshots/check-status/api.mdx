Once you start a snapshot load, you can check the load status using
the `/sys/storage/raft/snapshot-load` endpoint:

```shell-session
$ curl                                       \
   --request GET                             \
   --header "X-Vault-Token: ${VAULT_TOKEN}"  \
    ${VAULT_ADDR}/v1/sys/storage/raft/snapshot-load/<snapshot_id>
```
