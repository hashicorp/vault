### vault.identity.nonentity.active.monthly ((#vault-identity-nonentity-active-monthly))

| Metric type | Value    | Description                                                                                         |
|-------------|----------|-----------------------------------------------------------------------------------------------------|
| gauge       | non entities | The number of distinct non entities (across all namespaces) that created a token during the past month. |

Vault reports `vault.identity.nonentity.active.monthly` at the start of each month
when client counting is enabled.
