### vault.core.replication.dr.secondary ((#vault-core-replication-dr-secondary))

Metric type | Value   | Description
----------- | ------- | -----------
gauge       | boolean | Indicates whether the Vault node is a disaster recovery secondary

- A value of `1` indicates that the node **is** a disaster recovery secondary.
- A value of `0` indicates that the node **is not** a disaster recovery secondary.
