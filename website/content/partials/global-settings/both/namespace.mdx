**`[-namespace | -ns | VAULT_NAMESPACE] (string : <unset>)`**

</a>

Root namespace for the CLI command. Setting a default namespace allow relative
mount paths.

**Examples**:

- CLI flag: `-namespace "admin"`
- Environment variable: `export VAULT_NAMESPACE="admin"`
