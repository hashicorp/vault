**`[-log-format | VAULT_LOG_FORMAT] (enum : standard)`**

</a>

Format of log data:

- **`standard`** - Write log data as basic text.
- **`json`** - Write log data as JSON.

**Examples**:

- CLI flag: `-log-format json`
- Environment variable: `export VAULT_LOG_FORMAT=json`
