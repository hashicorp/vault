## Active Directory secrets engine ((#ad-secrets-engine))

| Announced | Expected end of support | Expected removal |
| :-------: | :---------------------: | :--------------: |
| MAR 2023  | CY24 Q3                 | CY25 Q1

The Active Directory secrets engine is deprecated. We recommend
[migrating your Active Directory calls](/vault/docs/secrets/ad/migration-guide)
to use the [LDAP Secrets Engine](/vault/docs/secrets/ldap) with the `ad`
[schema](/vault/api-docs/secret/ldap#schema).
