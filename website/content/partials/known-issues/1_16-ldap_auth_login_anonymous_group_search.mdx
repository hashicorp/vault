### Error logging in with LDAP auth method when anonymous group search is enabled

#### Affected versions

- 1.16.0

#### Issue
Depending on their LDAP configuration, some customers may encounter an error
when attempting to login with ldap auth method when anonymous group search is
enabled. See this [issue](https://github.com/hashicorp/vault/issues/26183) for more details.

#### Workaround

There is no workaround.
