### Error logging in with LDAP auth method

#### Affected versions

- 1.16.0

#### Issue
Depending on their LDAP configuration, some customers may encounter an error when attempting to login with ldap auth method.
Active Directory users are affected by this bug.
See this [issue](https://github.com/hashicorp/vault/issues/26171) for more details.

This issue was resolved in `1.16.1`.

#### Workaround

There is no workaround.
