**`-plugin-version (string : "")`**

Configures the semantic version of the plugin to use. If unspecified, implies the
built-in or any matching unversioned plugin that may have been registered.

The new version will not start running until the mount is 
[reloaded](/vault/docs/commands/plugin/reload).

**Example**: `-plugin-version "v1.12.0+builtin.vault"`
