**`-max-lease-ttl (string : "")`**

The maximum lease TTL for this secrets engine. If unspecified, this defaults to
the Vault server's globally configured maximum lease TTL. This value is allowed to
override the server's global max TTL; it can be longer or shorter. Uses 
[duration format strings](/vault/docs/concepts/duration-format).

**Example**: `-max-lease-ttl "30m"`
