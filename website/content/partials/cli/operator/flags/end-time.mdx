**`-end-time (string : "")`**

End of report period. Defaults to end of the current month.

The output shows the exact time range being reported, which may not match the 
input parameters if a full month is not available, or if the available reports 
are a subset of the months requested.

**Example**: `-end-time "2020-01"`
