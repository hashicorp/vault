**`-pgp-key (string : "")`**

Comma-separated list of paths to files on disk containing public PGP keys OR 
a comma-separated list of Keybase usernames using the format `keybase:<username>`. 
When supplied, Vault will encrypt and base64-encode the generated root token with 
the given public key. The number of entries must match `-key-shares`, unless 
`-stored-shares` are used.

**Example**: `-pgp-key "keybase:hashicorp"`
