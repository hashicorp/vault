**`-log-level (string : "info")`**

Log verbosity level. Supported values (in order of detail) are `"trace"`, 
`"debug"`, `"info"`, `"warn"`, and `"error"`. These are not case sensitive.

You can also specify this via the `VAULT_LOG_LEVEL` environment variable.

**Example**: `-log-level "debug"`
