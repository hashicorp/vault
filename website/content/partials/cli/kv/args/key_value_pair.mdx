**`key_value_pair (string : <required>)`**

A key and data value in one of the following forms:

Format                        | Description
----------------------------- | -----------
`<key_name>=<value>`          | Defines a single key and explicit value
`<key_name>=@<json_file_path> | Defines a single key and reads value data from a JSON file
`<key_name> -`                | Defines a single key and reads value data from `stdin`
`@<json_file_path>            | Reads one or more key and value pair from a JSON file


**Example**: `prod=5678`
