**`-delete-version-after (string : "-1ns")`**

Specifies the length of time before a version is deleted. If not set, Vault uses
the backend's configured `delete-version-after`. 

This value cannot be greater than the backend's `delete-version-after`. The 
`delete-version-after` is specified as a numeric string with a suffix like 
"30s" or "3h25m19s".

**Example**: `-delete-version-after "30s"`
