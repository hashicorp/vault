**`-delete-version-after (string : "0s")`**

Time in `<number>[s|m|h|d]` format to wait before automatically deleting data
versions going forward. The `-delete-version-after` value cannot be greater than
the `delete_version_after` value in the plugin configuration. Use `"0s"` to
disable automatic version pruning.

**Example**: `-delete-version-after "30s"`
