**`-cas (int : 0)`**

Perform a check-and-set operation against the provided version before applying
the patch to avoid race conditions and potential data loss for targeted
key/value data. Vault ignores the `-cas` flag when `-method` is `rw`.

`cas` value | Description
----------- | -----------
0           | Always patch the data
N           | Patch data if the current version is `N`

**Example**: `-cas 1`
