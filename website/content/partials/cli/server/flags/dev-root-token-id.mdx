**`-dev-root-token-id (string: "")`**

Initial root token. This only applies when running in `dev` mode. You can also
specify this via the `VAULT_DEV_ROOT_TOKEN_ID` environment variable.

<Note>
  
The token ID should not start with the `s.` prefix.

</Note>

**Example**: `-dev-root-token-id my-root-token`
