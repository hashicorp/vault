**`-log-file (string : "")`**

The absolute path where Vault should save log messages in addition to other,
existing outputs like journald / stdout. Paths that end with a path separator use
the default file name, `vault.log`. Paths that do not end with a file extension
use the default `.log` extension. You can also specify this via the
`VAULT_LOG_FILE` environment variable.

| `log-file`              | Full log file           | Rotated log file                    |
|-------------------------|-------------------------|-------------------------------------|
| `/var/log`              | `/var/log/vault.log`    | `/var/log/vault-{timestamp}.log`    |
| `/var/log/my-diary`     | `/var/log/my-diary.log` | `/var/log/my-diary-{timestamp}.log` |
| `/var/log/my-diary.txt` | `/var/log/my-diary.txt` | `/var/log/my-diary-{timestamp}.txt` |

**Example**: `-log-file /var/log`
