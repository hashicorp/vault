**`-strict-host-key-checking (string : "")`**

Value to use for the SSH configuration option "StrictHostKeyChecking". The
default is ask. You can also specify this via the 
`VAULT_SSH_STRICT_HOST_KEY_CHECKING` environment variable.

**Example**: `-strict-host-key-checking no`
