**`-host-key-hostnames (string : "*")`**

List of hostnames to delegate for the CA. The default value allows all
domains and IPs. This is specified as a comma-separated list of values. You can
also specify this via the `VAULT_SSH_HOST_KEY_HOSTNAMES` environment variable.

**Example**: `-host-kegiy-hostnames example.com,192.168.1.1`
