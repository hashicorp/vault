# Vault release notes

- **Version**: 1.20.x
- **GA date**: 2025-06-11

@include 'release-notes/intro.mdx'

## Executive summary

Vault 1.20.0 revamps user experience related to critical functions such as
billing and visibility. The result is more intuitive and predictable pricing,
better transparency around features and capabilities, and streamlined workflows
for key operations like secret recovery, plugin management, and cryptography. 


Vault Enterprise 1.20.0 streamlines the user experience, improves visibility and
transparency around pricing. The latest version of Vault also introcudes new
capabilities related to cryptography, secret recovery, and provides enhanced
ecosystem integrations for centralizing secrets.


### Highlights

- Improves cost-saving by streamlining billing transparency and contextualizing
  price increases .

- Simplifies and enhances the user experience by improving namespace selection,
  providing a customizable login function, and releasing a new secret recovery
  function.

- Enhances and expandss secure integrations and reduces friction on plugin
  distribution by allowing customers to import key value-compatible secrets from
  AWS, Azure, and GCP.

- Adds SCEP protocol support in Vault PKI with verified integrations for Azure
  Intune and JAMF for certificate automation that reduces operational burdens
  such as outages or security breachs due to certificate expiry, and reduces IT
  footprint by eliminating the need for alternate PKI solutions explictly for
  SCEP integration.

- Enhances integrations with existing workflows and tools by reducing friction
  on plugin distribution.
  
- Provides a secure, single source of truth by allowing customers to import key
  value-compatible secrets from AWS, Azure, and GCP.

- Enhances resilience by providing reliability improvements, control over
  traffic flows, and the ability to ensure fairness of Vault consumption across
  users and applications.   

- Better auditability and visibility into audit logs, certificates, Vault
  feature usage, and opinionated suggestions for improving Vault usage including
  benchmarking that supports migrating from Consul to integrated storage.
