# policy list

List the names of the policies that are installed on the Vault server.
<CodeBlockConfig hideClipboard>

```shell-session
$ vault policy list [flags]

$ vault policy list [-help | -h]
```

</CodeBlockConfig>

`policy list` lists the names of the policies that are installed on the Vault 
server.

</Tip>
