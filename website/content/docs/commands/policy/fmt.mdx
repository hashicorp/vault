# policy fmt

Format a local policy file to the policy specification.

<CodeBlockConfig hideClipboard>

```shell-session
$ vault policy fmt [flags] <path>

$ vault policy fmt [-help | -h]
```

</CodeBlockConfig>

## Description

`policy fmt` formats a local policy file to the policy specification. This
command will overwrite the file at the given `path` with the properly-formatted
policy file contents.
