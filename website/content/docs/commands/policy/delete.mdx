# policy delete

Delete a policy.

<CodeBlockConfig hideClipboard>

```shell-session
$ vault policy delete [flags] <name>

$ vault policy delete [-help | -h]
```

</CodeBlockConfig>

## Description

`policy delete` deletes the policy named `name` in the Vault server. Once the 
policy is deleted, all tokens associated with the policy are affected
immediately.

You cannot delete the `"default"` or `"root"` policies. These are built-in 
policies.

<Tip title="Related API endpoints">

DeletePolicy - [`DELETE: /sys/policy/:name`](/vault/api-docs/system/policy#delete-policy)

</Tip>
