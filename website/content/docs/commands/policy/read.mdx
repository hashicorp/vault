# policy read

Prints the contents and metadata of the Vault policy.

<CodeBlockConfig hideClipboard>

```shell-session
$ vault policy read [flags] <name>

$ vault policy read [-help | -h]
```

</CodeBlockConfig>

## Description

`policy read` prints the contents and metadata of the Vault policy named `name`. 
If the policy does not exist, Vault returns an error is returned.

<Tip title="Related API endpoints">

ReadPolicy - [`GET: /sys/policy/:name`](/vault/api-docs/system/policy#read-policy)

</Tip>
