# `kv destroy`

<Note title="Version-specific command">

The `kv destroy` command only works with key/value v2 plugins.

</Note>

Permanently delete data from the provided key/value secrets path.

<CodeBlockConfig hideClipboard>

```shell-session
$ vault kv destroy [flags] <secret_path>

$ vault kv destroy [-help | -h]
```

</CodeBlockConfig>



## Description

`kv destroy` permanently deletes the targeted version of data on a given path in
key/value v2 secrets plugins. `kv destroy` fails gracefully and takes no action
if the secret path is invalid.

<Tip title="Related API endpoints">

  DestroySecretVersions - [`PUT: /:secret-mount-path/destroy/:path`](/vault/api-docs/secret/kv/kv-v2#destroy-secret-versions)

</Tip>

## Examples

Destroy version 11 of the key `creds`:

```shell-session
$ vault kv destroy -mount=secret -versions=11 creds
Success! Data written to: secret/destroy/creds
```
