# `kv delete`

Soft delete data from the provided key/value secrets path.

<CodeBlockConfig hideClipboard>

```shell-session
$ vault kv delete [flags] <secret_path>

$ vault kv delete [-help | -h]
```

</CodeBlockConfig>



## Description

`kv delete` soft deletes the targeted version of data on a given path in
key/value v2 secrets plugins. **`kv delete` permanently deletes  data for
key/value version 1 plugins**.

<Tip title="Related API endpoints">

- DeleteSecret (v1) - [`DELETE: /secret/:path`](/vault/api-docs/secret/kv/kv-v1#delete-secret)
- DeleteSecretLatestVersion (v2) - [`DELETE: /:secret-mount-path/data/:path`](/vault/api-docs/secret/kv/kv-v2#delete-latest-version-of-secret)
- DeleteSecretVersion (v2) - [`POST: /:secret-mount-path/delete/:path`](/vault/api-docs/secret/kv/kv-v2#delete-secret-versions)

</Tip>

## Examples

### Key/value v2

Delete the latest version of data for the key `creds`:

```shell-session
$ vault kv delete -mount=shared creds

Success! Data deleted (if it existed) at: shared/creds
```

Delete version 11 of data for the key `creds`:

```shell-session
$ vault kv delete -mount=shared -versions "11" creds

Success! Data deleted (if it existed) at: shared/data/creds
```


### Key/value v1

Permanently delete data for the key `seeds`:

```shell-session
$ vault kv delete -mount=shared seeds

Success! Data deleted (if it existed) at: shared/seeds
```
