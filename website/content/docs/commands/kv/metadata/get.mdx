# kv metadata get

<Note>

This is a [KV version 2](/vault/docs/secrets/kv/kv-v2) secrets engine command,
and not available for Version 1.

</Note>

Retrieves the metadata of the versioned secrets at the given key name.

<CodeBlockConfig hideClipboard>

```shell-session
$ vault kv metadata get [flags] <key>

$ vault kv metadata get [-help | -h]
```

</CodeBlockConfig>

`kv metadata get` retrieves the metadata of the versioned secrets at the given
key name. If no key exists with that name, Vault returns an error.

<Tip title="Related API endpoints">

ReadSecretMetadata (v2) - [`GET: /:secret-mount-path/metadata/:path`](/vault/api-docs/secret/kv/kv-v2#read-secret-metadata)

</Tip>
