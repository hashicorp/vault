# `kv enable-versioning`

Upgrade a key/value v1 plugin to v2.

<CodeBlockConfig hideClipboard>

```shell-session
$ vault kv enable-versioning [flags] <mount_path>

$ vault kv enable-versioning [-help | -h]
```

</CodeBlockConfig>



## Description

`kv enable-versioning` upgrades an existing key/value v1 plugin to v2 by turning
on versioning.

## Examples

Turn on versioning for a `kv` version 1 plugin enabled at `shared`.

```shell-session
$ vault kv enable-versioning shared

Success! Tuned the secrets engine at: shared/
```
