# operator key-status

Provides information about the active encryption key.

<CodeBlockConfig hideClipboard>

```shell-session
$ vault operator key-status [flags]

$ vault operator key-status [-help | -h]
```

</CodeBlockConfig>

## Description

`operator key-status` provides information about the active encryption key.
Specifically, the current key term and the key installation time.

## Command arguments

None.

## Command options

None.

## Command flags

None.

## Examples

Get the key status:

```shell-session
$ vault operator key-status
Key Term          2
Install Time      01 Jan 17 12:30 UTC
Encryption Count  4494
```
