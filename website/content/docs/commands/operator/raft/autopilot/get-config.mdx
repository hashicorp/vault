# operator raft autopilot get-config

Returns the configuration of the autopilot subsystem under integrated storage.

<CodeBlockConfig hideClipboard>

```shell-session
$ vault operator raft autopilot get-config [flags]

$ vault operator raft autopilot get-config [-help | -h]
```

</CodeBlockConfig>

## Description

`operator raft autopilot get-config` returns the configuration of the autopilot
subsystem under integrated storage.

<Tip title="Related API endpoints">

GetConfig - [`GET: /sys/storage/raft/autopilot/configuration`](/vault/api-docs/system/storage/raftautopilot#get-configuration)

</Tip>
