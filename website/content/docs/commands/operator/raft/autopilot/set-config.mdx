# operator raft autopilot set-config

Modify the configuration of the autopilot subsystem under integrated storage.

<CodeBlockConfig hideClipboard>

```shell-session
$ vault operator raft autopilot set-config [flags]

$ vault operator raft autopilot set-config [-help | -h]
```

</CodeBlockConfig>

## Description

`operator raft autopilot set-config` modifies the configuration of the autopilot
subsystem under integrated storage.

<Tip title="Related API endpoints">

SetConfig - [`POST: /sys/storage/raft/autopilot/configuration`](/vault/api-docs/system/storage/raftautopilot#set-configuration)

</Tip>
