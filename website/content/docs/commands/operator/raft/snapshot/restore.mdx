# operator raft snapshot restore

Installs the provided snapshot, returning the cluster to the state defined in it.

<CodeBlockConfig hideClipboard>

```shell-session
$ vault operator raft snapshot restore [flags] <snapshot-file>

$ vault operator raft snapshot restore [-help | -h]
```

</CodeBlockConfig>

## Description

`operator raft snapshot restore` installs the provided snapshot, returning the 
cluster to the state defined in it.
