# operator raft snapshot inspect

Inspects a snapshot file.

<CodeBlockConfig hideClipboard>

```shell-session
$ vault operator raft snapshot inspect [flags] <snapshot-file>

$ vault operator raft snapshot inspect [-help | -h]
```

</CodeBlockConfig>

## Description

`operator raft snapshot inspect` inspects a snapshot file taken from a Vault 
Raft cluster and prints a table showing the number of keys and the amount of 
space used.
