# operator raft snapshot save

Saves a snapshot of the current state of the Raft cluster into a file.

<CodeBlockConfig hideClipboard>

```shell-session
$ vault operator raft snapshot save [flags] <snapshot-file>

$ vault operator raft snapshot save [-help | -h]
```

</CodeBlockConfig>

## Description

`operator raft snapshot save` saves a snapshot of the current state of the Raft 
cluster into a file.

You can use the snapshot to restore Vault to the point in time when a snapshot 
was taken.

<Note>

Snapshot is not supported when Raft is used only for `ha_storage`.

</Note>
