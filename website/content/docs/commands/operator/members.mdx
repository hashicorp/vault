# operator members

Provides the details of all the nodes in the cluster.

<CodeBlockConfig hideClipboard>

```shell-session
$ vault operator members [flags]

$ vault operator members [-help | -h]
```

</CodeBlockConfig>

## Description

`operator members` provides the details of all the nodes in the cluster.

## Command arguments
