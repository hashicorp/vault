# version

Prints the Vault version.

<CodeBlockConfig hideClipboard>

```shell-session
$ vault version

$ vault version [-help | -h]
```

</CodeBlockConfig>

## Description

`version` prints the version of the Vault CLI. This does not print the target 
Vault server version.

This command is aliased as `-v` or `--version`.

## Command arguments

None.

## Command options

None.

## Command flags

None.

## Global flags

None.

## Examples

Print the version of the Vault CLI:

```shell-session
$ vault version
Vault v1.18.1 (f479e5c85462477c9334564bc8f69531cdb03b65), built 2024-10-29T14:21:31Z
```
