# version

The `version` command prints the Vault version:

```shell-session
$ vault version
Vault v1.2.3
```

It can also be printed by adding the flags `--version` or `-v` to the `vault` command:

```shell-session
$ vault -v
Vault v1.2.3, built 2022-05-03T08:34:11Z
```
