# plugin register

<CodeBlockConfig hideClipboard>

```shell-session
$ vault plugin register [flags] <type> <name>

$ vault plugin register [-help | -h]
```

</CodeBlockConfig>

## Description

`plugin register` registers a new plugin in Vault's plugin catalog. You must 
include the plugin's type of `"auth"`, `"database"`, or `"secret"` in the command.

<Tip title="Related API endpoints">

RegisterPlugin - [`POST: /sys/plugins/catalog/:type/:name`](/vault/api-docs/system/plugins-catalog#register-plugin)

</Tip>

## Examples

Register a plugin:

```shell-session
$ vault plugin register \
    -sha256=d3f0a8be02f6c074cf38c9c99d4d04c9c6466249 \
    auth my-custom-plugin
Success! Registered plugin: my-custom-plugin
```

Register a plugin with custom args:

```shell-session
$ vault plugin register \
    -sha256=d3f0a8be02f6c074cf38c9c99d4d04c9c6466249 \
    -args=--with-glibc \
    -args=--with-curl-bindings \
    auth my-custom-plugin
```
