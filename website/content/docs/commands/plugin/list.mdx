# plugin list

Lists all available plugins in the plugin catalog.

<CodeBlockConfig hideClipboard>

```shell-session
$ vault plugin list [flags] <type>

$ vault plugin list [-help | -h]
```

</CodeBlockConfig>

## Description

`plugin list` lists all available plugins in the plugin catalog. You can use 
this command alone or include the plugin's type of `"auth"`, `"database"`, or 
`"secret"` in the command.

<Tip title="Related API endpoints">

- ListPlugins - [`GET: /sys/plugins/catalog`](/vault/api-docs/system/plugins-catalog#list-plugins)
- ListPluginsAuth - [`GET: /sys/plugins/catalog/auth`](/vault/api-docs/system/plugins-catalog#list-plugins-1)
- ListPluginsDatabase - [`GET: /sys/plugins/catalog/database`](/vault/api-docs/system/plugins-catalog#list-plugins-1)
- ListPluginsSecret - [`GET: /sys/plugins/catalog/secret`](/vault/api-docs/system/plugins-catalog#list-plugins-1)

</Tip>

### Deprecation status column

As of 1.12, all builtin plugins will have an associated Deprecation
Status. This status will be reflected in the `Deprecation Status` column, seen
below. All non-builtin plugins will show a `Deprecation Status` of "n/a".

## Usage

The following flags are available in addition to the [standard set of
flags](/vault/docs/commands) included on all commands.

### Output options

- `-format` `(string: "table")` - Print the output in the given format. Valid
  formats are "table", "json", or "yaml". This can also be specified via the
  `VAULT_FORMAT` environment variable.

### Command options

- `-detailed` `(bool: false)` - Print detailed information such as version and
deprecation status about each plugin.


## Examples

List all available plugins in the catalog.

```shell-session
$ vault plugin list
Name                                 Type        Version
----                                 ----        -------
alicloud                             auth        v0.13.0+builtin
# ...
```

List all available secret plugins in the catalog:

```shell-session
$ vault plugin list database
Name            Version
----            -------
ad              v0.14.0+builtin
alicloud        v0.13.0+builtin
# ...
```

List all available database plugins in the catalog:

```shell-session
$ vault plugin list database
Name                                 Version
----                                 -------
cassandra-database-plugin            v1.13.0+builtin.vault
# ...
```

List detailed plugin information:

```shell-session
$ vault plugin list -detailed
Name        Type    Version                  Container    Deprecation Status
----        ----    -------                  ---------    ------------------
alicloud    auth    v0.12.0+builtin          false        supported
app-id      auth    v1.12.0+builtin.vault    false        pending removal
# ...
```
