# plugin runtime info

Displays information about a plugin runtime in the catalog with the given name.

<CodeBlockConfig hideClipboard>

```shell-session
$ vault plugin runtime info [flags] <type> <name>

$ vault plugin runtime info [-help | -h]
```

</CodeBlockConfig>

`plugin runtime info` displays information about a plugin runtime in the
catalog with the given name. If the plugin runtime does not exist, Vault returns
an error.

Support for runtime types is currently limited to `container`. Support for the
`container` runtime is currently limited to Linux.

## Examples

Display information about a plugin:

```shell-session
$ vault plugin runtime info -type=container runc
Key              Value
---              -----
cgroup_parent    n/a
cpu_nanos        0
memory_bytes     0
name             runc
oci_runtime      runc
type             container
```
