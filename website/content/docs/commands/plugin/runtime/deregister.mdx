# plugin runtime deregister

Deregister an existing plugin runtime in the catalog with the given name.

<CodeBlockConfig hideClipboard>

```shell-session
$ vault plugin runtime deregister [flags] <name>

$ vault plugin runtime deregister [-help | -h]
```

</CodeBlockConfig>

## Description

`plugin runtime deregister` deregisters an existing plugin runtime in the
catalog with the given name. Returns an error if a registered plugin references
the named plugin runtime or the named runtime does not exist.

Support for runtime types is currently limited to `container`. Support for the
`container` runtime is currently limited to Linux.

RemovePluginRuntime - [`DELETE: /sys/plugins/runtimes/catalog/:type/:name`](/vault/api-docs/system/plugins-runtimes-catalog#remove-plugin-runtime-from-catalog)

</Tip>
