# plugin deregister

Deregisters a plugin from Vault's plugin catalog.

<CodeBlockConfig hideClipboard>

```shell-session
$ vault plugin deregister [flags] <type> <name>

$ vault plugin deregister [-help | -h]
```

</CodeBlockConfig>

## Description

`plugin deregister` deregisters an existing plugin from Vault's plugin catalog. 
If the plugin does not exist, Vault does not return an error. You must include
the plugin's type of `"auth"`, `"database"`, or `"secret"` in the command.

<Tip title="Related API endpoints">

RemovePlugin - [`DELETE: /sys/plugins/catalog/:type/:name?version=:version`](/vault/api-docs/system/plugins-catalog#remove-plugin-from-catalog)

</Tip>
