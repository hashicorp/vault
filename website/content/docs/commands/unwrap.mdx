# unwrap

Unwraps a wrapped secret from Vault by the given token.

<CodeBlockConfig hideClipboard>

```shell-session
$ vault unwrap [flags] <token>

$ vault unwrap [-help | -h]
```

</CodeBlockConfig>

## Description

`unwrap` unwraps a wrapped secret from Vault by the given token. The
result is the same as the `vault read` operation on the non-wrapped secret. If
no token is given, the data in the currently authenticated token is unwrapped.

## Examples

Unwrap the data in the cubbyhole secrets engine for a token:

```shell-session
$ vault unwrap 3de9ece1-b347-e143-29b0-dc2dc31caafd
```

Unwrap the data in the active token. First, authenticate using a Vault token:

```shell-session
$ vault login 848f9ccf-7176-098c-5e2b-75a0689d41cd
```

Then, unwrap the secret:

```shell-session
$ vault unwrap # unwraps 848f9ccf...
```
