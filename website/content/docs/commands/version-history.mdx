# version-history

Prints the version history of the target Vault server.

<CodeBlockConfig hideClipboard>

```shell-session
$ vault version-history

$ vault version-history [-help | -h]
```

</CodeBlockConfig>

## Description

`version-history` prints the historical list of installed Vault versions in 
chronological order.

<Tip title="Related API endpoints">

ReadVersionHistory - [`LIST: /sys/version-history`](/vault/api-docs/system/version-history)

</Tip>
