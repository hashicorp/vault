# namespace lookup

Retrieve information about the namespace.

<CodeBlockConfig hideClipboard>

```shell-session
$ vault namespace lookup [flags] [path]

$ vault namespace lookup [-help | -h]
```

</CodeBlockConfig>

## Description

`namespace lookup` retrieves information about the namespace.

<Tip title="Related API endpoints">

ReadNamespace - [`GET: /sys/namespaces/:path`](/vault/api-docs/system/namespaces#read-namespace-information)

</Tip>

## Examples

Get information about the current namespace:

```shell-session
$ vault namespace lookup
```

Get information about a particular namespace (for example, `ns1/ns2/`):

```shell-session
$ vault namespace lookup -namespace=ns1 ns2
```
