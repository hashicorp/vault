# namespace lock

Lock a namespace and all descendants.

<CodeBlockConfig hideClipboard>

```shell-session
$ vault namespace lock [flags] [path]

$ vault namespace lock [-help | -h]
```

</CodeBlockConfig>

## Description

Locks a namespace and all descendants.

<Tip title="Related API endpoints">

LockNamespace - [`POST: /sys/namespaces/api-lock/lock/:subpath`](/vault/api-docs/system/namespaces#lock-namespace)

</Tip>

## Examples

Lock the current namespace and all descendants:

```shell-session
$ vault namespace lock
```

Lock a child namespace and all of its descendants (for example, `ns1/ns2/`):

```shell-session
$ vault namespace lock ns1/ns2
```
