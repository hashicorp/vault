# namespace unlock

Unlock a namespace and all descendants.

<CodeBlockConfig hideClipboard>

```shell-session
$ vault namespace unlock [flags] [path]

$ vault namespace unlock [-help | -h]
```

</CodeBlockConfig>

## Description

Unlock a namespace and all descendants.

<Tip title="Related API endpoints">

UnlockNamespace - [`POST: /sys/namespaces/api-lock/unlock/:subpath`](/vault/api-docs/system/namespaces#unlock-namespace)

</Tip>

## Examples

Unlock the current namespace and all descendants, with unlock key:

```shell-session
$ vault namespace unlock -unlock-key=<key>
```

Unlock the current namespace and all descendants (from a root token):

```shell-session
$ vault namespace unlock
```

Unlock a child namespace and all of its descendants (for example, `ns1/ns2/`):

```shell-session
$ vault namespace unlock -unlock-key=<key> ns1/ns2
```
