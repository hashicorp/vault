# namespace list

List all enabled child namespaces.

<CodeBlockConfig hideClipboard>

```shell-session
$ vault namespace list [flags]

$ vault namespace list [-help | -h]
```

</CodeBlockConfig>

## Description

`namespace list` lists all enabled child namespaces.
