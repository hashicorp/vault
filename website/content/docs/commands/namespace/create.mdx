# namespace create

Create a child namespace.

<CodeBlockConfig hideClipboard>

```shell-session
$ vault namespace create [flags] <path>

$ vault namespace create [-help | -h]
```

</CodeBlockConfig>

## Description

`namespace create` creates a child namespace. Vault will create the namespace 
relative to the namespace provided in either the `VAULT_NAMESPACE` environment 
variable or `-namespace` CLI flag.

<Tip title="Related API endpoints">

CreateNamespace - [`POST: /sys/namespaces/:path`](/vault/api-docs/system/namespaces#create-namespace)

</Tip>

## Examples

Create a namespace at the path `ns1/` with no custom metadata:

```shell-session
$ vault namespace create ns1/
```

Create a child namespace from a parent namespace (for example, `ns1/ns2/`):

```shell-session
$ vault namespace create -namespace=ns1 ns2
```

Create a namespace at the path `ns1/` with multiple custom metadata keys:

```shell-session
$ vault namespace create -custom-metadata=foo=abc -custom-metadata=bar=123 ns1/
```
