# namespace delete

Delete an existing namespace.

<CodeBlockConfig hideClipboard>

```shell-session
$ vault namespace delete [flags] <path>

$ vault namespace delete [-help | -h]
```

</CodeBlockConfig>

## Description

`namespace delete` deletes an existing namespace. Vault will delete the namespace 
relative to the namespace provided in either the `VAULT_NAMESPACE` environment 
variable or `-namespace` CLI flag.

<Tip title="Related API endpoints">

DeleteNamespace - [`DELETE: /sys/namespaces/:path`](/vault/api-docs/system/namespaces#delete-namespace)

</Tip>

## Examples

Delete the namespace at path `ns1/`:

```shell-session
$ vault namespace delete ns1/
```

Delete a namespace namespace from a parent namespace (for example, `ns1/ns2/`):

```shell-session
$ vault namespace delete -namespace=ns1 ns2
```
